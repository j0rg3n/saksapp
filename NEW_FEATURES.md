#  New features


## Eventuelt-saker

SaksApp: eventuelt -saker skal være ordentlige saker, bare med egen status siden de blir lagt til i referatet og ikke agendaen.

## SaksApp: innfør styrelogg, hendelser som så kobles på sakene.

- Kan fortsatt lage hendelser i en sak.
- Kan lage hendelser uten sak.
- Kan lage hendelser direkte fra WhatsApp eller telegram ved å binde til WhatsApp bruker (med utelukkende identitet tilgang).
- Kan merke hendelse med flere saker.
- Hendelse kan være avvik, tiltak, og andre kategorier som vi finner ut av etterhvert.
- Behandle møte notater som logg, men med spesiell status: "referat" og "vedtak" og "oppfølging"

## Avviks- og tiltakslogg for HMS

Innfør avviks- og tiltakslogg som viser alle avvik og tiltak. Vil typisk ikke inneholde møter, siden man ikke fatter tiltak der. Men kan ha referanser til saker, selvsagt.

## Whatsapp integrasjon

Saksappbot til WhatsApp i Styrelogg-gruppen


### Speile loggen begge veier
Når speilet, tagg med WhatsApp meldings ID og oppdater ved redigering. Evt. post by versjon hvis redigering feiler, med link til gammel: som et svar.
